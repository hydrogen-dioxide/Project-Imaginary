Sad Birthday (v2)
Time
1 s
Memory
256 MB

Daniel's birthday is coming! You, as a friend of him, decided to celebrate Daniel's 16th birthday. Unfortunately, there are too much test and exam. During exams, you cannot celebrate his birthday because you need to do revision (of course)! Therefore you decided to write a programme to see if you can celebrate his 16th birthday.

INPUT

The first line contains \(Yb\), \(Mb\), \(Db\), separated by spaces. Indicating Daniel's 16th birthday.

The second line contains an integer \(N\), indicating the number of exams.

In the following \(2N\) lines, \(i\) ranging from 1 to \(N\).

The \((2(i-1)+i)\)th line contains \(Yis\), \(Mis\), \(Dis\), separated by spaces. Indicating the starting date of the ith exam.

The \(2i\)th line contains \(Yie\), \(Mie\), \(Die\), separated by spaces. Indicating the ending date of the \(i\)th exam.

OUTPUT

If none of the exam overlap Daniel's 16th birthday, output "Happy Birthday!".

If any of the exam overlap Daniel's 16th birthday, output "Sad Birthday :(".
