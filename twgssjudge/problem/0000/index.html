Printing Pages
Time
1s
Memory
1GB
One day, a mysterious explosion occur; and all the printers around the city had broken! You, as a computer enthusiast, decided to make a printer program, to reinvent the printers. Can you successfully be the hero and change the world?

INPUT

The first line consists of two integers \( P, N \), the number of pages in the document to be printed and the length of the input string \( S \).

The second line consists of the input string \( S \). It is guranteed that it contains only ascii codes. However, any printable symbols in ascii, not restricted other than numbers, ',', ';' and '-', except '\n' the nextline symbol, might exist in the input string \( S \).

OUTPUT

Output a single integer, the number of pages that should actually be printed. If the format of the input string \( S \) or the page number(s) is incorrect, output -1 (or the printer will be crashed!).

SAMPLE TESTS

# Input Output
1 10 7
1,2,5-7 		5
Page 1, 2, 5, 6, 7, total of 5 pages, will be printed.
2 10 12
01, 2; 5 - 7 		5
Explanation same as sample test 1. Prevailing zeros 01 and spaces may exist in the string \( S \).

CONSTRAINTS

\( 1 \leq P \leq 10^9 \)

\( 1 \leq N \leq 10^5 \)

All symbols of string \( S \) is in ascii character set and printable.

'\n', the nextline symbol, doesn't exist in \( S \).
